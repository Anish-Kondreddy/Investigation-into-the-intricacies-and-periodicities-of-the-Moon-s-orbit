# Investigation-into-the-intricacies-and-periodicities-of-the-Moon-s-orbit
Utilised Python to process 100+ years of data, applying FFT and time-series forecasting. Benchmarked numerical integrators (Euler, Velocity-Verlet, Runge Kutta 4), employed finite-difference and PDE discretisation techniques to optimise stability and improve computational efficiency.
